Minha página HTML

HTML é uma linguagem de marcação padrão para páginas web.
Com html você pode criar o seu próprio site.


O que é o HTML?

  • HTML é uma linguagem de marcação web responsável por estruturar um site
  • Ela é formada por uma série de elementos, como tags e atributos
  • Os elementos html informam ao navegador como exibir o conteúdo

Quais plataformas são mais utilizadas para se programar em HTML?

  1. Visual Studio Code (VS Code);
  2. Sublime Text;
  3. Atom.

Para saber mais sobre essa temática, basta acessar w3schools.
